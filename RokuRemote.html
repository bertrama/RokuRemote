Instructions:
This web page has been designed to allow the Google Chrome web browser to control the Roku media player. If you do not have either Google Chrome or a Roku media player then this web page will be of no use to you. You can download Chrome web browser from Google and the Roku media player is available for purchase from Roku.com.

Before you start using this web page, get the IP address of your Roku player. This can be found in the Roku under the Roku Player Settings by selecting "player info." Once you have the IP address, type it into the text box at the top of the Roku Remote web page. Then click the "Set IP Address" button. If you set it correctly then you should now be able to use the remote buttons to control your Roku.

The "Set Icon Size" will let you set the percentage size you want the icon symbols to be re-sized at. By default, the icons are set to 100% but you can make them larger or smaller to better fit your screen.

The "Text/Icons" button can toggle the remote control buttons from being symbols to being text buttons. Pressing the "Text/Icons" button can then switch it back from "Text/Icons" back to icons.

Clicking on the remote buttons should have the same effect as using the Roku IR (InfraRed) remote buttons. More information about the Roku remote control can be found on the Roku Support web page.

If you have the cursor in focus in the "Remote Text Box" then you can use the keyboard as a Roku remote. A map between keyboard keys and Roku remote functions are provided at the end of the instructions.

If the "Search Text Box" has a cursor in focus then letters, numbers, space and backspace will be sent to the Roku. This is useful if a Roku channel is currently on a Search screen. Instead of using the on-screen keys, you should be able to use the Search Text Box to type on the computer keyboard normally.

There is also an option to display a "Search Keypad" which might be helpful for some tablet users. If you are having trouble using the "Search Text Box" then you may want to consider using the Search Keypad option instead.

The "Remote Text Box" command keys are as follows:

  • 1 - Go back
  • 3 - Home
  • 2 - Up
  • Q - Left
  • E - Right
  • S - Down
  • W - Select/OK
  • A - Instant Replay
  • D - Options/Info
  • Z - Rewind
  • X - Play/Pause
  • C - Fast Forward
